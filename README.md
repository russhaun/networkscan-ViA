# Networkscan
> Raspberry Pi local network device scanner - open port, default password check for multiple vendors.
### Requirements
Raspberry Pi Model 3 or 4
### Installation
Nmap:
```
# sudo apt-get install python3-nmap -y
```
Bluez-tools:
```
# sudo apt-get install bluez-tools -y
```
